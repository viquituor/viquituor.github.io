Projeto Final do Estágio na Compass UOL

No projeto final do estágio na Compass UOL, desenvolvi um pipeline completo de engenharia de dados e um dashboard interativo para visualização dos insights gerados. O objetivo foi criar uma solução eficiente para ingestão, processamento e análise de dados em AWS.

Automação e Monitoramento: Criação de scripts em Python para automação das tarefas de processamento, garantindo maior eficiência e confiabilidade.

Dashboard Interativo: Desenvolvido no Amazon QuickSight, permitindo a visualização dinâmica dos dados, com gráficos e métricas detalhadas para facilitar a tomada de decisões.

Segurança e Eficiência: Aplicação de boas práticas para garantir integridade e escalabilidade na manipulação dos dados.

Esse projeto consolidou minha experiência prática em big data, computação em nuvem e engenharia de dados, além de fortalecer minhas habilidades em automação e visualização de dados.
Tecnologias utilizadas: Python, Apache Spark, AWS S3, AWS Glue, AWS Lambda, Amazon QuickSight, Docker.

Sistema de Gestão Jurídica

Desenvolvi um sistema jurídico completo com foco na gestão de rotinas de um escritório de advocacia.

Trata-se de um produto real, iniciado como um trabalho acadêmico da disciplina de Banco de Dados, e posteriormente expandido e documentado nas disciplinas de Análise e Modelagem de Sistemas, Modelagem de Processos e Desenvolvimento de Sistemas.

O sistema foi construído do zero, com base em requisitos fornecidos diretamente por um Product Owner, sem o apoio de uma equipe de desenvolvimento. A stack utilizada inclui Node.js e Express no backend, React no frontend e MySQL, com modelagem feita no BRModelo e MySQL Workbench.

Módulos desenvolvidos:
|Advogados | Clientes | Contratos | Processos| Pagamentos | Agenda | Prazos Processuais|

A aplicação possui navegação organizada por módulos, formulários para cadastro e edição, e listagens de dados dinâmicas.

O backend fornece uma API completa e o frontend garante uma experiência clara e funcional.

O projeto demonstra minha capacidade de integrar conhecimento técnico com análise de requisitos reais, resultando em uma solução prática e aplicável ao mundo jurídico.
