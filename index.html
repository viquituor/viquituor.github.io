AWS Glue: Aprendi a usar o AWS Glue para realizar tarefas de ETL (extração, transformação e carga), facilitando a integração e processamento de grandes volumes de dados. A capacidade de automatizar fluxos de trabalho de dados me permitiu entender melhor a arquitetura de dados em ambientes de nuvem.

AWS Athena: Utilizei o AWS Athena para realizar consultas SQL em dados armazenados no Amazon S3. Essa experiência me ensinou a analisar rapidamente grandes conjuntos de dados sem a necessidade de gerenciar a infraestrutura subjacente, tornando a análise de dados mais acessível e eficiente.

AWS S3: Trabalhar com o Amazon S3 me proporcionou uma compreensão sólida do armazenamento em nuvem. Aprendi a organizar e gerenciar dados, garantindo que estivessem sempre disponíveis e seguros, o que é essencial para qualquer projeto de ciência de dados.

AWS Lambda: Explorar o AWS Lambda me ensinou a criar funções serverless para executar código em resposta a eventos, permitindo uma abordagem mais escalável e econômica para o desenvolvimento de aplicações.

MySQL: Trabalhar com bancos de dados MySQL me permitiu entender melhor o gerenciamento de dados relacionais. Aprendi a criar e otimizar consultas SQL, além de projetar esquemas de banco de dados para suportar aplicações eficazes.

Spark: O uso do Apache Spark me possibilitou realizar processamento de dados em larga escala. Aprendi a criar pipelines de dados eficientes e a aplicar técnicas de análise avançadas para extrair insights valiosos de conjuntos de dados complexos.

Python: Durante o estágio, aprofundei meu conhecimento em Python, uma linguagem fundamental para ciência de dados e desenvolvimento. Aprendi a utilizar bibliotecas como Pandas e NumPy, além de aplicar práticas de programação limpa e eficiente.

Docker: A experiência com Docker me ensinou a criar ambientes de desenvolvimento consistentes e a facilitar a implantação de aplicações. Compreendi a importância da contêinerização na modernização de fluxos de trabalho de desenvolvimento e operações.
