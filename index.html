EXPERIENCIAS

COMPASS UOL SCHOLLARSHIP - AWS DATA CLOUD ENGINEERING

Durante meu estágio como Engenheiro de Dados na Compass, tive a oportunidade de trabalhar com um conjunto de ferramentas da AWS, o que ampliou minha visão sobre o processamento e gerenciamento de dados em larga escala. Trabalhei com tecnologias como AWS Glue, Amazon S3, Athena e Lambda, criando pipelines de dados eficientes para automatizar processos de ETL e análise de grandes volumes de dados. O uso do AWS Glue me ajudou a criar processos de extração, transformação e carga (ETL) de dados de forma automática, enquanto o Amazon S3 foi essencial para o armazenamento e organização desses dados. Com o AWS Lambda, desenvolvi funções serverless para automatizar respostas a eventos em tempo real, eliminando a necessidade de gerenciar servidores. Além disso, a experiência com o Apache Spark me permitiu processar grandes conjuntos de dados com eficiência, aproveitando o poder do processamento distribuído. Essa experiência consolidou meu conhecimento técnico em engenharia de dados e me preparou para trabalhar em ambientes que demandam soluções escaláveis e integradas para grandes volumes de dados.
